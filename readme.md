≡ƒö¼ **Research question:** *what happens when the attacker reasons, plans, and learns?*

The field of **Agentic VAPT** sits at the intersection of large language model reasoning, sequential decision-making, and offensive security. My work builds an end-to-end framework where an LLM-driven agent performs reconnaissance, vulnerability analysis, and exploitation under formal governance ΓÇö extended to the OT/IIoT environments where conventional automated tooling breaks down.
